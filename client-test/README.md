# WindNet Client Test (Godot UI & Multiplayer)

This is a standalone Godot project designed to verify that your WindNet AWS backend deployed successfully. It includes a functional User Authentication interface and a real-time Multiplayer Synchronization test.

> **⚠️ CRITICAL ARCHITECTURE NOTE**
> The scripts provided in this project are baseline examples designed to prove the cloud infrastructure is routing data correctly. 
> 
> The included multiplayer script (`multi_player_test.gd`) utilizes a 10Hz network tick rate combined with Godot native vector extrapolation—perfect for RPGs or top-down adventures. If you are building a fast-paced action game, this baseline will not be sufficient. You are expected to customize or replace the client-side interpolation and tick rates to fit your specific game's requirements.

## 🧪 Test 1: Cognito Authentication
Verify that your user database and secure token generation are functioning.

**Setup Instructions:**
1. Open this project in the Godot Engine.
2. Locate the `login.gd` script.
3. Replace the placeholder values at the top with the IDs generated by your AWS deployment:
   ```gdscript
   COGNITO_USER_POOL_ID = "your-pool-id"
   COGNITO_CLIENT_ID = "your-client-id"
   COGNITO_ENDPOINT = "your-cognito-endpoint"
   ```
4. Press Play in the Godot editor to test creating a user and logging in.

## ⚔️ Test 2: AppSync Multiplayer
Verify that your GraphQL endpoints and WebSocket (WSS) subscriptions are passing real-time data between clients.

**Setup Instructions:**
1. Locate `multi_player_test.gd` and the WindNet Autoload script (`subscriber.gd`).
2. Replace the placeholder AWS details with your deployed AppSync credentials:
   ```gdscript
   appsync_url = "[your-api-id.appsync-api.your-region.amazonaws.com/graphql](https://your-api-id.appsync-api.your-region.amazonaws.com/graphql)"
   appsync_realtime_domain = "your-api-id.appsync-realtime-api.your-region.amazonaws.com"
   api_key = "your-api-key" 
   # Note: For production, swap the API key for Cognito JWT authentication
   ```
3. Go to **Debug > Run Multiple Instances** in the Godot top menu and select **2 Instances**.
4. Press Play. Move your character in Window 1 and watch the coordinates natively extrapolate in Window 2.

## 🛠️ What This Project Does
* **Sign Up / Confirmation:** Creates a user in AWS Cognito and handles email verification.
* **Login:** Authenticates the user and retrieves a secure web token (JWT), stored in a Singleton.
* **Real-Time WebSockets:** Establishes a persistent `graphql-ws` connection to AWS, managing the base64 authentication handshake.
* **Chronological Syncing:** Demonstrates sequence-number tracking to eliminate out-of-order packet rubber-banding.
